# Tiny Swords RTS

一款即時戰略塔防遊戲，使用 [Phaser 3](https://phaser.io/) 引擎開發。玩家操控騎士陣營，建立基地、生產士兵，抵禦從四面八方進攻的哥布林波次敵人。撐過 20 波攻擊即為勝利。

**線上試玩：** https://easylive1989.github.io/tiny_sword/

## 遊戲操作

### 基本操作

| 操作 | 說明 |
|------|------|
| 左鍵點擊單位 | 選取單位，底部顯示資訊面板 |
| 左鍵點擊建築 | 選取建築，可生產對應兵種 |
| 左鍵點擊空地 | 取消選取 |
| 左鍵拖曳 | 框選範圍內的多個單位 |
| 右鍵點擊地面 | 命令已選取的單位移動到該位置 |
| 右鍵點擊敵人 | 命令戰士攻擊目標敵人 |
| 方向鍵 / 滑鼠邊緣 | 移動視角 |
| Ctrl + 數字鍵 (1-9) | 建立控制編隊 |
| 數字鍵 (1-9) | 選取控制編隊 |
| M 鍵 | 切換小地圖顯示 |
| 左鍵點擊小地圖 | 快速移動視角到該位置 |

### 遊戲流程

1. **建造建築** — 點擊城堡上的「Build」按鈕，選擇要建造的建築類型，綠色表示可放置，紅色表示不可放置，左鍵確認放置，右鍵取消
2. **生產兵種** — 選取兵營/射箭場/修道院，點擊「Train」按鈕，等待倒計時後兵種出現在建築旁
3. **控制部隊** — 使用左鍵拖曳框選多個單位，或使用 Ctrl + 數字鍵建立控制編隊，方便快速調度
4. **監控戰場** — 使用小地圖快速查看敵我位置，點擊小地圖可快速移動視角
5. **防�禦敵人** — 戰士和弓箭手會自動偵測並攻擊附近的敵人，僧侶會自動治療受傷友軍，也可以手動右鍵指定攻擊目標
6. **撐過 20 波** — 每 60 秒會有一波哥布林從地圖邊緣進攻，難度逐波遞增，全部擊退即為勝利

### 勝負條件

- **勝利** — 撐過全部 20 波敵人攻擊，並消滅所有殘餘敵人
- **敗北** — 主堡 (Castle) 被敵人摧毀

### 遊戲規模

- **地圖大小** — 80×48 格 (5120×3072 像素，每格 64px)
- **波次數量** — 20 波，分為 4 個階段：學習期 (1-5)、成長期 (6-10)、快速期 (11-15)、極限挑戰 (16-20)
- **地圖變化** — 10 種隨機地圖，包含不同的地形分布（樹木、水域、山地）

## 開發

### 安裝與啟動

```bash
npm install
npm run dev          # 啟動 live-server (port 8081)
```

瀏覽器開啟 http://localhost:8081 即可遊玩。

### 執行測試

```bash
npm test             # 執行 Playwright 測試
```

### Lint

```bash
npm run lint         # ESLint 檢查 src/
```
