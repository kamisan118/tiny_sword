# i18n 設計

## 概述

為遊戲加入英文/中文雙語支援，在 Landing 畫面左上角提供語系切換按鈕。

## 架構

單一 `src/i18n/i18n.js` 模組，包含：
- 英文/中文翻譯物件
- `t(key, params?)` 函式，支援 `{placeholder}` 插值
- `setLocale(locale)` / `getLocale()` 管理語系
- 語系偏好存在 `localStorage`，預設 `'en'`

## 語系切換行為

- Landing 左上角方形按鈕（ui_btn_sq_blue），顯示 `EN` / `中`
- 切換後存入 localStorage，重啟 LandingScene
- 遊戲中不提供切換，需回到 Landing 才能切換

## 翻譯對照表

約 20 個 key，涵蓋所有 UI 文字（按鈕、HUD、建築名稱、訓練按鈕、勝敗訊息、載入畫面）。
